RegisterRepository

RegisterRules
Registers fluent rule classes with the container.
RegisterSession(ContainerBuilder)
Registers ISession with the container. By default session is registered as an instance per lifetime scope and is autowired with a IDependencyResolver.

RegisterSessionFactory(ContainerBuilder)
Registers ISessionFactory with the container. Requires that IRuleRepository is registered with the container. By default session factory is registered as a single instance and is autowired with a IDependencyResolver.
RegisterSessionFactory(ContainerBuilder, FuncIComponentContext, ISessionFactory)
Registers ISessionFactory with the container. By default session factory is registered as a single instance and is autowired with a IDependencyResolver.
